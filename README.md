**Uncle Doge Site**

*** How to test locally: ***

Requirements:
NodeJS

1.Go inside project folder
2.Run "npm install"
3.Run "npm run dev"


*** How to deploy to server: ***

1.Add folder to server directory
2.Run command line on server and go inside project folder
2.Run "npm install"
3.Run "npm run export"
4.Run "npm run start"
7.Done. Your website is deployed
